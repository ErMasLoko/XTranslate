h3. XTranslate

The main goal of this extension is providing easy work in the web in different languages.
All you need is make text selection on the page and press hotkey (Ctrl+Shift+X by default), then translated text immediately showing up beside.
The extension already supports "Google":http://translate.google.com/ and "Yandex":http://translate.yandex.ru/.

h3. Install

Download and execute *.oex-file from "Github":/extensible/XTranslate/tree/master/releases or "Opera":https://addons.opera.com/en/addons/extensions/details/xtranslate/

See more about Opera extensions "here":http://www.opera.com/addons/extensions/develop/.
